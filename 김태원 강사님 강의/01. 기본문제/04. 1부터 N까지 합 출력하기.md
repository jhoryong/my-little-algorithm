### 문제
자연수 N이 입력되면 1부터 N까지의 합을 출력하는 프로그램을 작성하세요.

### 내 풀이
```js
// sol1
function solution(N) {
  let result = 0;
  for (let i = 1; i <= N; i++){
    result += i;
  }
  return result;
}

// sol2 
function solution(N) {
  return (1 + N) * (N / 2);
}
```
