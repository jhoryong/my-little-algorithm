### 문제
한 개의 문자열을 입력받아 해당 문자열에 알파벳 대문자가 몇 개 있는지 알아내는 프로그램 을 작성하세요.


### 내 풀이
```js
function solution(str) {
  let arr = str.match(/[A-Z]/g);
  return arr.length;
}
```

### 정답
```js
function solution(s){         
    let answer=0;
    for(let x of s){
        //let num=x.charCodeAt();
        //if(num>=65 && num<=90) answer++;
        if(x===x.toUpperCase()) answer++; 
    }

    return answer;
}
```
